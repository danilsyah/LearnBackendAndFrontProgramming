# Pendahuluan 
Pembelajaran menggunakan Lumen sebagai backend dan VUE+Vuetify sebagai Frontend.
# Requirements
Proses pembelajaran ini membutuhkan beberapa hal yang perlu dipersiapkan oleh peserta ada yang bersifat wajib dan opsional, yaitu :
* PHP >= 7.2.5 [wajib]
* Composer 1.9.1
* NPM 6.14.4
* MySQL / Maria DB
* Editor bebas, tetapi direkomendasikan Visual Studio Code (VS Code). Bila  VS Code yang digunakan, installkan extension Laravel Blade Snippets, Vetur, vuetify-vscode
